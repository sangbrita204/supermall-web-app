
# SuperMall Web Application

## Description
A web portal that allows merchants to advertise and sell products
while enabling customers to browse shops, offers, and categories.

## Modules
- Admin Login
- Create & Manage Shops
- Category & Floor Wise Details
- Offer Management
- Shop & Offer Listing

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore
- GitHub Pages
